# Projet : Application mobile intelligente d’aide au transport public

## 1. Introduction
Cette application mobile a pour objectif de **faciliter les déplacements en transports publics** pour les utilisateurs.  
Elle permet de saisir un **point de départ** et un **point d’arrivée**, puis de proposer les **options de transport disponibles** avec :  
- Horaires  
- Tarifs  
- Durée estimée  

Le système recommande ensuite le **trajet optimal** en utilisant des **techniques de Machine Learning**, tout en prenant en compte les préférences et l’historique des utilisateurs.

## 2. Objectifs du projet
- Aider les utilisateurs à **trouver le meilleur trajet rapidement**.  
- Fournir les informations détaillées sur les **moyens de transport**.  
- Proposer des **recommandations intelligentes** selon les critères de temps, coût et confort.  
- Assurer une **interface intuitive et sécurisée** pour tous les utilisateurs.
