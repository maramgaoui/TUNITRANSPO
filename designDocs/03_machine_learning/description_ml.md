# Description du Machine Learning

## 1. Introduction
Le Machine Learning (ML) est utilisé pour **analyser les données de transport** et **recommander le meilleur trajet** aux utilisateurs selon différents critères (temps, coût, confort).  

L’objectif est de **prédire et optimiser les trajets** en fonction des préférences des utilisateurs et des données en temps réel sur les transports.

## 2. Données utilisées
- Horaires des bus, métros, trains et louages  
- Tarifs des trajets  
- Durées estimées et historiques de trajets  
- Feedback utilisateur (trajets favoris, trajets suivis, évaluations)  

## 3. Rôle du Machine Learning
- Calculer un **score optimal** pour chaque trajet possible.  
- Adapter les recommandations aux **préférences personnelles** de l’utilisateur.  
- Améliorer les recommandations au fil du temps grâce aux **données collectées**.
