# Fonctionnalités liées au Machine Learning

## 1. Recommandation personnalisée
- Propose le **meilleur trajet** selon les critères de l’utilisateur :  
  - Temps total  
  - Coût  
  - Confort (nombre de correspondances, mode de transport préféré)  

## 2. Prise en compte des données en temps réel
- Ajuste les recommandations si des **retards ou indisponibilités** sont détectés.  

## 3. Historique et apprentissage
- Analyse les trajets précédents de l’utilisateur pour **améliorer les futures recommandations**.  

## 4. Adaptation continue
- Les modèles ML s’adaptent à l’évolution des **habitudes des utilisateurs** et des **données de transport**.
